ExternalUserFunction

MIL CoPilot Sample Workspace

This workspace shows how to use an external function inside MIL CoPilot.

Using the blob module, we locate a nut or bolt in the image. For each blob occurrence, we use our external user function, MoveChild, defined in python to move a pre-allocated child buffer to the location of the occurrence. The external user function gets the location of the bounding box from the blob results, and move and resizes the child buffer. The child buffer is then displayed.
