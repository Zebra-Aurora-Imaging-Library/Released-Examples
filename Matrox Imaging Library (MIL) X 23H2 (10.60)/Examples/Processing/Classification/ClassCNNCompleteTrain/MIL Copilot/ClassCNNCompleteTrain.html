Class CNN Complete Train

MIL CoPilot Sample Workspace

This workspace sets up all the necessary objects for performing a CNN training. To obtain a trained context, perform the following:

  1. Press Train in the Image Classification ribbon.
  2. Select the Training Dataset, associated Classifier context, and Dev Dataset, if the default ones are not correct.
  3. Press OK.

After the training completes, the trained classifier context will be restored along with the predicted dev and train datasets. You can perform a predict operation using the trained classifier context.
